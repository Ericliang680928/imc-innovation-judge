## ⏭️ 從哪裡接續?

### 雲端部署的 4 個剩餘步驟

#### 1️⃣ 建立 GitHub Repo
- 開啟 → https://github.com/new
- Repository name:`imc-innovation-judge`
- 設定 **Public**
- ⚠️ 初始化選項**全部不勾**(因為本機已有 commit)
- 點 **Create repository**

#### 2️⃣ Push 到 GitHub
```bash
cd "D:\IMC\2026-創新比賽\imc_judge_app"
git remote add origin https://github.com/【你的用戶名】/imc-innovation-judge.git
git push -u origin main
```
> 第一次會跳 GitHub 登入視窗,用瀏覽器登入授權

#### 3️⃣ 連接 Streamlit Cloud
- 開啟 → https://share.streamlit.io
- **Sign up → Continue with GitHub**
- 點 **New app**
- 選 repo:`imc-innovation-judge`
- Main file path:`app.py`
- App URL:`imc-judge`(會變成 `imc-judge.streamlit.app`)
- 點 **Deploy!**

#### 4️⃣ 取得公開 URL
- 部署完成後會有類似:`https://imc-judge.streamlit.app`
- 把 URL 更新到 `README.md` 的「線上版」段落
